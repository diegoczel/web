CSS Selectors

We can divide CSS selectors into five categories

  • Simple selector (select based on tag, id, class, universal, grouping)
  • Combinator selectors (select based on a specific relationship between them)
  • Pseudo-class selectors (select based on a certain state)
  • Pseudo-elements selector (select a part of an element)
  • Attribute selectors (select based on an attribute or attribute value)
